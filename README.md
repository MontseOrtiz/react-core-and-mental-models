# React: Fundamentos y Modelos Mentales

Este repositorio contiene mis apuntes estructurados mientras estudio React.  
El objetivo no es solo aprender la sintaxis, sino comprender los modelos mentales que hacen que React funcione: renderizado, estado, reconciliación, hooks y herramientas modernas.

---

## 🎯 Objetivo del repositorio

- Reforzar conceptos fundamentales de React
- Construir modelos mentales sólidos
- Documentar aprendizajes técnicos
- Servir como base de conocimiento a largo plazo
- Registrar mini proyectos prácticos del curso

## 🧠 Filosofía de aprendizaje

El enfoque de estos apuntes es entender cómo funciona React internamente y por qué se toman ciertas decisiones al desarrollar.

Aprender React no es memorizar hooks, sino comprender:

- Cómo renderiza
- Cuándo re-renderiza
- Cómo maneja el estado
- Cómo optimizar correctamente
